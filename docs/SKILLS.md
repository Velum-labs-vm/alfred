# ALFRED Skills

Skill packs live under `src/skills` and follow:

- `index.ts`
- `handlers.ts`
- `permissions.ts`
- `README.md`

Available packs:

- voice
- calendar
- contacts
- sms
- notifications
- memory
- automation
- connectors
- ambient
- developer

All skills are designed for local-first behavior and should be toggleable via settings integration.
